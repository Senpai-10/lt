# **lt**

Simple todo list manager

## Default config

config file: `.config/lt/config.toml`

```toml
# This option is used
# if no --file provided
default_database_file = "todo.db"

# Length of genrated id for tasks
id_length = 3

# Format of date used when listing a task
date_format = "%Y-%m-%d %I:%M:%S %P"
```
